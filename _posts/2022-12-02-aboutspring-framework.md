### Spring Framework?

- Spring makes it easy to create Java enterprise applications.

스프링 공식 레퍼런스에 들어가보면 Overview에 나오는 첫 줄이다. 

즉, 스프링은 자바 어플리케이션을 만들기 쉽게 해주는 프레임워크이다. 

- Spring Framework 6.0 부터는 Java17 이상이 필요하다.
- 애플리케이션의 요구 사항에 따라, 다양한 종류의 아키텍처를 생성할 수 있는 유연성을 통해 엔터프라이즈 환경에서 Java 언어를 수용하는데 필요한 모든 것을 제공한다.
- 오픈소스

자바의 자바에 의한 자바를 위한 스프링이다☕️

### Design Philosophy

스프링 프레임워크의 기본 원칙

- 모든 수준에서 선택권을 제공한다.
- 다양한 관점을 수용한다. 유연성
- 강력한 이전 버전과의 호환성
- API 디자인 (직관적, 유지성이 좋은 API)
- 코드 품질에 대한 높은 기준
